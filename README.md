# Customer Segmentation Analysis

Customer segmentation project using K-Means, Agglomerative Clustering, and DBSCAN on the Mall Customers dataset.

## Overview

This project analyzes customer characteristics to identify meaningful customer groups. The data was cleaned and scaled before applying three clustering methods. Model results were compared using silhouette score and Davies–Bouldin index.

## Methods

- Data cleaning and preprocessing
- Feature scaling
- K-Means clustering
- Agglomerative clustering
- DBSCAN
- Silhouette score
- Davies–Bouldin index
- Cluster visualization

## Files

- `customer_segmentation_analysis.ipynb` – Complete analysis, code, visualizations, and results
